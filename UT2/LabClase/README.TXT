Proyecto: LabClases
Autores: David J. Barnes y Michael Kolling

Este proyecto ilustra la interacción entre objetos y examina código muy simple. (Capítulos 1 y 3).

Para utilizar este proyecto:
    
        - crear algunos objetos Estudiante
        - crear un objeto LabClase
        - añadir los estudiantes a la clase utilizando el método matricularEstudiante.
        - Escribir una lista de clase
