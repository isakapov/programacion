Projecto "figuras"
Authores: Michael Kolling y David J. Barnes

Este proyecto es parte del material del libro:

   "Objects First with Java - A Practical Introduction using BlueJ"
   2ª edition
   David J. Barnes y Michael Kolling
   Pearson Education, 2004

Es un proyecto muy simple para demostrar algunas características de los objetos.

Se pueden crear varias figuras: cuadrados, triángulos, círculos.
Estas figuras se dibujan en una pantalla (dentro de una ventana llamada "canvas" - lienzo)

Una vez creados, estos objetos se pueden manipular: cambiar su posición, tamaño y color. 

Este proyecto está diseñado como un primer ejemplo de programación orientada a objetos. Ilustra varios conceptos:

 - un proyecto Java (una aplicación) es una colección de clases
 - los objetos se crean a partir de las clases
 - a partir de una clase se pueden crear varios objetos
 - los objetos poseen operaciones (métodos)
 - las operaciones pueden tener parámetros
 - los parámetros tienen un tipo (al menos String e int)
 - los objetos poseen datos (atributos - campos)
 - las operaciones y los atributos son comunes a todos los objetos
 - los valores almacenados en los atributos pueden ser diferentes para cada objeto

El proyecto también muestra:

 - creación de objetos en BlueJ
 - invocación de métodos de manera interactiva
 - paso de parámetros

Un segundo proyecto "dibujo" añade una clase a las existentes en este proyecto. La nueva clase (Dibujo)
utiliza las figuras para realizar un dibujo. Puede ser utilizado para experimentar  con el código.
